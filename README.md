Niche: Daily Work Log & Report App

Niche is a productivity and reporting tool designed to help employees effortlessly log their daily tasks and automatically compile reports for supervisors or managers. It simplifies tracking and communication of daily progress in professional environments.

⸻

Key Features
	•	Daily Logging: Employees input daily work activities and notes.
	•	Auto-Compilation: The app compiles daily inputs into weekly/monthly reports.
	•	Report Delivery: Reports can be shared with managers via email or download.
	•	Secure Access: Each user has a secure login to manage their entries.
	•	User Dashboard: Personalized view of logs and report history.

⸻

Tech Stack
	•	Frontend: React / HTML / CSS / JavaScript
	•	Backend: Node.js / Express
	•	Database: MongoDB / Firebase / PostgreSQL
	•	Authentication: JWT / Firebase Auth
	•	Deployment: Vercel / Netlify / Render (based on your setup)
